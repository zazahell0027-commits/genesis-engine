# Comment verifier cette mise a jour

Cette branche sert a verifier que les retours suivants ont bien ete pris en compte dans la suite de la conception :

- clic droit sur un pays
- menu contextuel d'actions
- fin des boutons de guerre permanents
- recentrage sur la geopolitique
- interface plus propre
- francais partout

## Branche
`proposal/contextual-country-menu-v2`

## Fichier a verifier
`proposals/contextual-country-menu-v2.md`

## Commandes
```bash
git fetch origin
git checkout proposal/contextual-country-menu-v2
```

Ensuite tu ouvres le fichier de proposition dans VS Code et tu verifies que les demandes sont bien notees.
